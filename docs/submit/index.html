사이클 기획안
찍기 전에 먼저 설계합니다.

격주 사이클마다 만들 영상의 타깃, 주제, 구성을 정리해 제출해주세요. 제출 내용과 AI 검토 결과를 이메일로 보내드립니다.

이렇게 진행됩니다
  1. 제출할 사이클과 마감일을 먼저 확인합니다.
  2. 한 명의 타깃 시청자를 떠올리고, 영상 주제와 흐름을 구체적으로 적습니다.
  3. 제출하면 작성 내용 전문 · AI 검토 · 수정 링크 · 기획안 갤러리 링크가 이메일로 도착합니다.
  4. 같은 이메일과 사이클로 다시 제출하면 기존 기획안이 업데이트됩니다.
